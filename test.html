Форматы файлов

.DOC или .doc (от англ. document) - расширение имени файла, используемого для файлов, представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов без форматирования, однако позже стало использоваться для форматов с разметкой.

Portable Document Format ( PDF) - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда возможностей языка Portscript.
Документ PDF

.zip и хранит в сжатом виде один или несколько файлов, которое можно из него извлечь путем распаковки с помощью специальной
